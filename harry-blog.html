The Harry Character

Written by Dhara Pancholi

As a kid I always adored Harry Potter as super hero. I turned my fantasy to real, while making this character project. This process taught me all different aspects of creating the character model to animation. I have tried to portray my entire process of building this character to life.



#1. References used for the character.

#2. The base body model.

#3. Before and after images, using spotlight projection for sculpting in Zbrush.

#4. Parts of character to be combined

#5. Model ready for textures.

#6. Another hair version using XGen tool

#7. The color scheme was made using reference images. This is the baked textured version.

After rigging the character, it was animated. I used maya FX tool for the jitters and explosion.
